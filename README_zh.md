## 🚀 快速开始

### 安装依赖
```bash
pip install selenium python-docx keyboard
```

### 下载ChromeDriver
```bash
# macOS (使用 Homebrew)
brew install chromedriver

# 或手动下载
# https://chromedriver.chromium.org/
```

### 运行程序
```bash
python get.py
```

## 💻 使用方法

1. **启动程序** - 自动打开Chrome浏览器
2. **导航页面** - 手动访问需要保存的网页
3. **保存内容** - 按回车键保存当前页面
4. **批量操作** - 可连续保存多个页面
5. **导出文档** - 输入'q'退出并保存Word文档

### 交互命令
- `回车键` - 保存当前页面
- `s` - 显示保存状态
- `q` - 退出并保存文档

## 🔧 技术特点

- **真实浏览器环境**：使用Selenium WebDriver模拟真实用户操作
